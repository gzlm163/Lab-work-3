# Лабораторная работа: Класс «Квадратная матрица» с перегрузкой операций

## Автор
Гинжул Мария

## Цель
Освоение принципов объектно-ориентированного программирования в C++, включая перегрузку операций, обработку исключений, реализацию интерфейсов и паттерна «Прототип».

## Описание
Разработан класс «Квадратная матрица», поддерживающий основные математические операции и сравнения. Реализованы конструкторы, перегружены операторы, методы для работы с детерминантом и обратной матрицей, а также глубокое копирование через паттерн «Прототип». Для демонстрации создано тестовое приложение «Матричный калькулятор».
